Where to find your data

Every submission writes two things to OneDrive:

  1. A PDF report at Metfraa-EHS / <form-folder> / Reports / YYYY / MM /
  2. An appended row in _MasterLog.xlsx at Metfraa-EHS / <form-folder> /

Adding more inspectors / admins

To add an inspector to the dropdown, edit server/lib/forms-config.js → INSPECTORS array → redeploy.

To add an admin email, update the ADMIN_EMAILS environment variable in Render and redeploy.
